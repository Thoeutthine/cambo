# Python Algorithm Exercises Collection

A comprehensive collection of algorithm exercises designed to improve programming skills in Python. This repository contains 300 carefully curated problems across three difficulty levels.

## 📚 Overview

This repository provides a structured approach to learning algorithms and data structures through hands-on practice. Each exercise includes:
- Clear problem statement
- Input/output examples
- Solution template
- Test cases
- Detailed explanations

## 🎯 Exercise Categories

### Beginner Level (100 exercises)
- Basic programming concepts
- Simple loops and conditionals
- String manipulation
- Basic math operations
- Array/list operations

### Medium Level (100 exercises)
- Data structures (stacks, queues, trees)
- Sorting and searching algorithms
- Dynamic programming basics
- Graph algorithms
- String algorithms

### Hard Level (100 exercises)
- Advanced dynamic programming
- Complex graph algorithms
- Advanced data structures
- Optimization problems
- Mathematical algorithms

## 🚀 Getting Started

1. Clone this repository:
```bash
git clone https://github.com/Thoeutthine/cambo.git
cd cambo
```

2. Navigate to the exercises directory:
```bash
cd exercises
```

3. Choose your difficulty level and start solving!

## 🏃‍♂️ How to Use

1. Read the problem statement in each `.py` file
2. Implement your solution in the provided template
3. Run the test cases to verify your solution
4. Compare with the provided solution if needed

Each exercise file includes test cases. To run them:
